# packages/ui — Componentes compartidos

Biblioteca de componentes React usada por `apps/web` y `apps/admin`.

## Cuándo agregar algo aquí
Solo componentes 100% agnósticos de negocio (Button, Input, Modal, etc.)
que se usen en más de una app. Lógica de negocio va en cada app.
